## Cek sendiri eksperimen:

1.Buka file .pkt salinan di Packet Tracer.

2.Periksa VLAN tiap switch: Click switch → Config → VLAN.

3.Cek IP Phone: Click IP Phone → lihat Voice VLAN & IP.

4.Ping antar PC: PC → Desktop → Command Prompt → ping [IP PC lain].

5.Lihat trunk antar-switch: Click switch → Port Settings → pastikan trunk aktif & VLAN diizinkan.
